# Getting Started
Getting the code
```bash
git clone https://github.com/swftdev/base.git
```

After cloning change terminal location
into the folder
```bash
cd base
```

Installing Needed Node modules so the app
can run
```bash
npm install
```

Opening your editor
```bash
code .
```

Starting the server
```bash
npm start
```
